Remind Me Later API

Welcome to the Remind Me Later API. This API allows you to create and manage reminders.

API Endpoints:

Endpoint Method Description
/api/reminders/ GET List all reminders
/api/reminders/ POST Create a new reminder
/api/reminders/{id}/ GET Retrieve a specific reminder
/api/reminders/{id}/ PUT Update a specific reminder
/api/reminders/{id}/ DELETE Delete a specific reminder

Example Request:

POST /api/reminders/
Content-Type: application/json

{
    "date": "2025-05-15",
    "time": "14:30:00",
    "message": "Call the doctor",
    "reminder_method": "sms",
    "recipient": "+1234567890"
}

Example Response:

{
    "status": "success",
    "message": "Reminder created successfully",
    "data": {
        "id": 1,
        "date": "2025-05-15",
        "time": "14:30:00",
        "message": "Call the doctor",
        "reminder_method": "sms",
        "recipient": "+1234567890",
        "created_at": "2025-05-11T12:00:00Z",
        "is_sent": false
    }
}
